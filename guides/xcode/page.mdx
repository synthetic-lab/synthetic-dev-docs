# Xcode Intelligence with Synthetic

Synthetic is a fully compatible provider for Xcode Intelligence, new in Xcode 26, enabling
context-aware suggestions and completions right in your Xcode workflow!

<Alert className="mb-4">

## Setup

### Step 1: Add Synthetic as a Model Provider

First, add Synthetic as a new model provider in Xcode settings:

![Add Synthetic model provider in Xcode settings](/images/dev/docs/xcode_1_add_model.png)

### Step 2: Configure Provider Details

Copy and paste these values into Xcode:
