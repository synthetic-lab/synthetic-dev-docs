# OpenCode with Synthetic

Synthetic is a native provider for [OpenCode](https://opencode.ai/), the TUI coding assistant!

With Synthetic, you can use OpenCode with any [model we support](/docs/api/models).

## Install OpenCode

Follow the official guide to [Install OpenCode](https://opencode.ai/docs#install).

## Quick Setup

### 1. Open OpenCode in Your Terminal

```bash
opencode
```

### 2. Connect to Synthetic

Type `/connect`.

![/connect](/images/dev/docs/opencode/opencode-connect.png)

### 3. Choose "Synthetic"

Select **Synthetic** from the list of available providers.

![Select Synthetic](/images/dev/docs/opencode/opencode-syntheticprovider.png)

### 4. Enter Your Synthetic API Key

Enter your [API Key](/docs/api/overview).

<div className="pb-4">
  <ApiKeyField
    initResp={await getOrCreateApiKey()}
    wrapperClassName="border-gray-300 text-gray-900"
  />
</div>

### 5. Choose a Model

Select which model you want to use from our [available models](/docs/api/models).

<Alert className="mb-4">
  <InfoIcon color="#00bb00" />
  <AlertTitle>Synthetic Hosted Models</AlertTitle>
  <AlertDescription>
    We recommend Synthetic hosted models for maximum quality and performance.
  </AlertDescription>
</Alert>


![Choose the desired open-source model.](/images/dev/docs/opencode/opencode-modelselect.png)

### 6. You're All Set!

Finish the rest of the onboarding flow to get started with OpenCode!
