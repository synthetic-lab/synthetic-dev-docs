# Moltbot with Synthetic

Synthetic is a native provider for [Moltbot](https://www.molt.bot/), the self-hosted, agentic LLM bot!

With Synthetic, you can use Moltbot with any [model we support](/docs/api/models),
including our high quality self-hosted models.

## Install Moltbot

Follow the official guide to [Install Moltbot](https://docs.molt.bot/start/getting-started).

## Quick Setup

Getting started with Moltbot and Synthetic takes just a few simple steps:

### 1. Open Moltbot in Your Terminal

```bash
moltbot
```

### 2. Select an Onboarding Mode

Select **QuickStart**.

![Select an Onboarding Mode - QuickStart](/images/dev/docs/moltbot/moltbot-onboarding.png)

### 3. Choose "Synthetic"

Select **Synthetic** from the list of available providers.

![Select Synthetic](/images/dev/docs/moltbot/moltbot-provider.png)

### 4. Paste Your Synthetic API Key

Select **Synthetic API Key** and enter your <SubdomainLink href="/user-settings/api">API Key</SubdomainLink>.

<div className="pb-4">
  <ApiKeyField
    initResp={await getOrCreateApiKey()}
    wrapperClassName="border-gray-300 text-gray-900"
  />
</div>

![Enter your Synthetic API Key](/images/dev/docs/moltbot/moltbot-apikey.png)

### 5. Choose a Model

Select which model you want to use from our [available models](/docs/api/models).

<Alert className="mb-4">
  <InfoIcon color="#00bb00" />
  <AlertTitle>Synthetic Hosted Models</AlertTitle>
  <AlertDescription>
    <p>
      We recommend Synthetic hosted models for maximum quality and performance.
    </p>
  </AlertDescription>
</Alert>


![Choose the desired open-source model.](/images/dev/docs/moltbot/moltbot-model.png)

### 6. You're All Set!

Finish the rest of the onboarding flow to get started with Moltbot!
