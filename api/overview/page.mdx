## Supported Endpoints

### OpenAI-compatible endpoints:

  <strong className="font-semibold shrink-0 flex items-center">Base Url:</strong>
  <BaseUrlField wrapperClassName="m-0" />
</div>

- [**`/models`**](/docs/openai/models) - List all always-on models and any recently used on-demand models.
- [**`/chat/completions`**](/docs/openai/chat-completions) - Chat-based completions with conversation history
- [**`/completions`**](/docs/openai/completions) - Traditional text completions
- [**`/embeddings`**](/docs/openai/embeddings) - Transform text into vector embeddings.
